Multilingual TTS (Text-to-Speech) technology enables the conversion of written text into spoken language across multiple languages.
These systems facilitate efficient and inclusive communication, breaking down language barriers and enhancing accessibility for users worldwide.

Note:- You should have internet connection and text file should be well text-formatted.


To run:-

1.Clone this repository in VS Code.
2.Open New Terminal in VS Code and type command "pip install -r requirements.txt"(without double quotes) for the installation of libraries.
3. Now, Type command "Streamlit run app.py", allow the permission your system is asking for.
4. Application will be opened in the browser asking for you to select languages and upload file.
5. Wait for a while until it complete it's execution.
